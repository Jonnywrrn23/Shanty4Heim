# Sea Shanty 2 Sailing Mod

Plays the iconic Sea Shanty 2 from RuneScape when you dance on a boat in Valheim!

## Installation

Install with r2modman or manually place in BepInEx/plugins/

## Usage

1. Get on any boat
2. Use the dance emote
3. Enjoy the tunes!
